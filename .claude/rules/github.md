# GitHub rules

## No AI co-authorship

Never add an AI coding assistant as a co-author on any commit — no `Co-Authored-By:` trailer (or equivalent) naming Claude, Claude Code, Anthropic, Codex, OpenAI, GPT, Open Code, or any other AI tool/model/vendor. This applies to every commit in this repo, no exceptions unless the user explicitly asks for one in the moment.

Write commit messages as if the user authored them — no mention of AI assistance anywhere in the message, trailer, or commit body.
